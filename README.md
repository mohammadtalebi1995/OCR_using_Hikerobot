# OCR using Hikrobot Camera

This project demonstrates an Optical Character Recognition (OCR) system for reading English letters and numbers using an industrial vision setup.

The system captures images using a Hikrobot industrial camera and processes them to detect and recognize alphanumeric characters. The goal is to enable automated identification of codes, serial numbers, or printed labels in industrial environments.

## Features

- Image acquisition using Hikrobot industrial camera
- Detection of English characters and numbers
- OCR processing for alphanumeric recognition
- Designed for industrial inspection and identification tasks
- Suitable for reading serial numbers, product codes, or labels

## Technologies

- Industrial Camera: Hikrobot
- Image Processing
- Optical Character Recognition (OCR)
- Industrial Machine Vision

## Application

This system can be used in manufacturing and production lines for:

- Reading serial numbers
- Product identification
- Label verification
- Automated tracking systems

## Future Improvements

- Improve OCR accuracy under different lighting conditions
- Add real-time processing for production line integration
- Expand character recognition capabilities
